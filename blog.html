Questions and Answers

  1. Discuss the scope of var, let, and const.

    2. Ans: Const and let is block scoped whereas var is global/functional scoped. Global scope means the variables declared outside the function can be accessed anywhere globally. Block scope means the variables can;t be accessed anywhere outside the particular block.

  2. Tell us the use cases of null and undefined.
    3. Ans: Null is a primitive data type in JavaScript. Null means a variable that is assigned but has no value.
    Ex: var x = null;
    Undefined means the variable has been declared but it's value is not assigned.
    Ex: var x;
    Cases of undefined are stated below.
    • variable that is not initialized.
    • parameters that is not passed.
    • condition.
    • function with no return.
    • property that doesn't exists in an object.
    • accessing array elements outside of the index range.
    • deleting an element inside an array.
    • set a value directly to undefined.
  3. What do you mean by REST API?

    4. Ans: API stands for Application Programming Interface and REST stands for Representational State Transfer. A REST API is a process that uses HTTPS to access and use data. That data can be used to CRUDE data types, which refers to Create Read Update Delete.
